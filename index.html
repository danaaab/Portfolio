Spotify Playlist Generator

The Billboard to Spotify Playlist Generator is a web application that allows users to create a Spotify playlist based on the Billboard Hot 100 songs from a specific date. Users enter a date, and the app scrapes Billboard's website to fetch the top songs from that day, then adds them to a new private Spotify playlist.

Flash Card Project

This Project is a simple language learning tool built using Python and the Tkinter library. It helps users practice French vocabulary by displaying a French word on a flashcard, then flipping it after a few seconds to reveal the English translation. Users can mark words they know, and the program saves unfamiliar words for future practice.
